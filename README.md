## Description

E-Commerce Back End Application using Express.js API along with Sequelize, to interact with a MySql database.

## Installation

To install, run the following command:  
```
npm i
```


## Usage 

To run this program:
```
npm start
```


To seed the data:
```
npm run seed
```
