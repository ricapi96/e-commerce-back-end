## Installation

To install, run the following command:  
```
npm i, npm mysql2, and npm sequelize
```


## Usage 

To run this program:
```
npm start
```

To run the schema from MySQL:
```
source db/schema.sql;
```

To seed the data:
```
npm run seed
```
